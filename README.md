Ce projet consiste à définir une graphe de connaissances dans le domaine de films. 
Le plan initiale est de construire un pipeline qui permettra une exploitation des données par chatbot et en langage naturel.

L'architecture de repo :
- archive contient les fichiers compresses en format .gz
- data (repo locale conseille) contient les fichiers de archive décompresses
- bin contient des scripts
